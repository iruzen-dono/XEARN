
# 🚀 XEARN

XEARN est une plateforme panafricaine de **micro-revenus digitaux** permettant aux utilisateurs de gagner de l’argent via :
- le visionnage de publicités rémunérées,
- un système d’affiliation à deux niveaux,
- des tâches sponsorisées.

La plateforme est conçue pour être **accessible à tous**, depuis un simple téléphone avec connexion Internet.

---

## 📌 Objectif du projet
Permettre aux jeunes et adultes africains de générer des revenus en ligne de manière simple, transparente et évolutive, tout en offrant aux entreprises une solution efficace de visibilité digitale.

---

## 🌍 Public cible
- Toute personne disposant :
  - d’un smartphone,
  - d’une connexion Internet.
- Lancement initial : **Togo**
- Expansion : **Afrique de l’Ouest puis Afrique entière**

---

## ⚙️ Fonctionnement général

### 🔐 Inscription & accès
- Inscription via :
  - Email
  - Numéro de téléphone (OTP)
- Application **web responsive** (PC, tablette, mobile).
- Chaque utilisateur dispose d’un **dashboard personnel**.

### 🆓 Mode gratuit
- Accès aux tâches publicitaires.
- ❌ Pas de retrait.
- ❌ Pas d’affiliation.

### 💳 Activation du compte
- Paiement unique : **4 000 FCFA**
- Débloque :
  - les retraits,
  - le lien d’affiliation,
  - les commissions de parrainage.

---

## 🧠 Système de tâches
- Type initial : visionnage de publicités.
- Validation : **automatique via API**.
- Rémunération : **variable selon la tâche**.
- Historique visible dans le dashboard utilisateur.

---

## 🔗 Système de parrainage
- **2 niveaux** :
  - Niveau 1 : **40 %**
  - Niveau 2 : **10 %**
- Chaque utilisateur possède :
  - un lien unique,
  - un QR Code traçable.
- Commissions créditées automatiquement.

---

## 💰 Paiements & retraits

### Moyens de paiement supportés :
- Mobile Money :
  - Flooz
  - T-Money
  - MTN MoMo
  - Orange Money
- Cartes bancaires (Visa / MasterCard)
- PayPal

### Retraits :
- Minimum : **2 000 FCFA**
- Traitement : **automatique**
- Historique consultable.

---

## 📢 Publicité & monétisation

### Abonnement Pub Maker
- Prix : **5 000 FCFA / mois**
- Permet aux utilisateurs/entreprises de :
  - publier leurs publicités,
  - apparaître dans une catégorie dédiée.

### Autres sources de revenus
- Frais d’activation utilisateurs
- Options premium (à venir)
- Publicités sponsorisées

---

## 🛠️ Administration

### Dashboard Administrateur

### Rapports automatiques
- Utilisateurs actifs.
- Revenus.

- Volume de retraits.
